Whizball Remastered

April 2024 - Present

I am currently remaking the once-popular 2000s game, Whizball (see image to the left), using a C physics library and developing a website to play the game using HTML. I am completing this with a partner and we will employ pair programming and a scrum methodology, keeping track of tasks using a Kanban board. The original game can be found at Criobite's archive of the game.
